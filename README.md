# Learning Management System

A learning management app that uses **YouTube videos** as course content. Built with React and Vite.

## Features

- **Open access** – Browse and use all courses without signing in
- **Home** – Hero and featured courses
- **Course list** – All courses by category
- **Lesson view** – YouTube player, cheat sheets, progress tracking
- **Code playground** – Run JavaScript and HTML/CSS/JS
- **AI Doubts** – Chat-style doubt assistant
- **Quizzes & practice** – Per-course assessments

## Setup

```bash
cd learning-management-system
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Tech stack

- React 18, React Router 6, Vite 5
- CSS (no UI library)
